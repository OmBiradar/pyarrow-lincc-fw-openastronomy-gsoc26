# PyArrow improvements for Astronomy

This project aims to improve the performance of PyArrow to read astronomy data files
stored in Parquet format!

The tech stack related to this field is C++, Python and build systems.

Technical knowledge include multi processing, modern C++ and language bindings.

GSoC blog cum repository on improving the performance of PyArrow
on tasks related to astronomy data processing.

## Structure

This repository contains scripts and other items that will help
in testing the updated and upgraded features.

This repo **will** contain in future:

- my fork of PyArrow in which I will make changes to improve the performance.
- scripts to test the performance upgrades
- a small blog about the work during GSoC project time.
