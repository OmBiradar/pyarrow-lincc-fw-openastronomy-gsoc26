# 6th June 2026, A day without AI

The previous python scripts were generated with the help of AI.
Which didn't really perform as expected. So now I wrote a new
script, that does not use AI. I manually went and checked the
various documentations and internet discussions on which is
better and why. Still some things were left out even at the
end of the day.

## A brief

- The parquet file generation logic was adopted from the
  GitHub issue
- The file were generated and safely deleted using the python
  `os` module
- The file was loaded safely into RAM using the `vmtouch`
  while creating a daemon to maintain the file in the RAM.
- The file was unloaded by closing the daemon using `.pid`
  files.
- The overall workflow was run as a demo workflow, which
  was successful at each step.
